# 🐾 Cat Dog Image Classification CNN

An AI-powered image classification project that identifies whether an uploaded image is a **Cat 🐱** or a **Dog 🐶** using a Convolutional Neural Network (CNN) built with TensorFlow and Keras.

This project also includes a modern Tkinter GUI where users can upload images and receive real-time predictions with a clean and interactive interface.

---

## 🚀 Features

* 🧠 CNN-based image classification
* 🐱 Detects Cats and Dogs from uploaded images
* 🖼️ Image upload with preview
* 💻 Modern Tkinter GUI design
* 📊 Training and validation accuracy graphs
* ⚡ Built using TensorFlow & Keras
* 🎨 Beginner-friendly and interactive UI
* 🤖 Friendly AI disclaimer messages

---

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/hadhiya-mn/Cat-Dog-Image-Classification-CNN.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Cat-Dog-Image-Classification-CNN
```

### 3️⃣ Install Dependencies

```bash
pip install tensorflow pillow matplotlib numpy
```

---

## ▶️ Run the Application

### Train Model

```bash
python classifier.py
```

### Run GUI Application

```bash
python predict.py
```

* Model Type: Convolutional Neural Network (CNN)
* Image Size: 128x128
* Binary Classification:

  * Cat = 0
  * Dog = 1
